Documentation for Marvel Template

Hello! Thank you for purchasing Marvel template. Below you will find a guide how to use it.

What's included

Within your downloaded zip archive you'll find the following directories and files:


html_full/

In html_full/ directory you'll find ready-to-use project files.

  • images/ folder contains image resources
  • scripts/ folder contains theme JS files
  • styles/ folder contains theme CSS files
  • vendor/ folder contains vendor files (a.k.a plugins, 3d-party libs)

html_seed/

In html_seed/ directory you'll find a seed project for in-depth customization. It includes Gulp file, Bower, Nunjucks files which can be used to quickly re-compile a theme’s HTML, CSS and JS. For more information about Nunjucks templating language please refer to Official Nunjucks documentation.

To use seed project you’ll need to have Node.js installed on your machine.
